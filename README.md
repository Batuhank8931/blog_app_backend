# Blog sitesi sunucu API;

## Çalıştırmak için;

- Proje klasörüne gidin:
  `cd blog_app_backend/`
  
- Composer gerekliliklerini indirin:
   `composer install`
  
- Testleri koşmak için:
  `php artisan test`
 
- Sunucuyu ayağa kaldırmak için:
  `php artisan serve`

  sunucu çalışırken [http://127.0.0.1:8000] adresinde çalışacaktır.
